Ask @lineage in GitHub Copilot Chat

Type @lineage in GitHub Copilot Chat to query your loaded graph. The assistant uses 9 dedicated tools — it answers from your actual data, never from general knowledge.

@lineage what schemas are loaded?
@lineage find tables with Employee in the name
@lineage what does HumanResources.Employee depend on?
@lineage trace 3 levels upstream from Sales.SalesOrderDetail
@lineage which objects have more than 10 connections?

Requires GitHub Copilot extension & VS Code 1.95+. Tools are inactive when no graph is loaded.

Copilot Chat (@lineage)

Ask questions about your loaded graph in GitHub Copilot Chat. The assistant answers using your actual data — schemas, tables, dependencies, DDL — never from general knowledge.

📁

Data Sources

Import from .dacpac files or directly from SQL Server, Azure SQL, Fabric DW, and Synapse databases. External tables shown as ⬡ nodes, with virtual nodes for OPENROWSET file paths and cross-database references.

🔍

Search & Navigate

Search and navigate objects with autocomplete. Schema-based color coding with interactive minimap.

🔌

Trace & Find Path

Trace upstream and downstream dependencies with sibling filtering, or find the shortest path between any two nodes.

📊

Graph Analysis

Detect islands, hubs, orphans, longest paths, and circular dependencies at a glance.

📄

DDL & Search

Click any node to view its DDL with full syntax highlighting. Full-text search across all SQL bodies.

📦

Export & Configure

Export the lineage graph to Draw.io. Custom parse rules, layout, and trace settings. Adapts to all VS Code themes.
